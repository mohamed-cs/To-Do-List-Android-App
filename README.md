# To-Do-List-Android-App
<p>A simple to-do list application written in Java.</p>
<h2>Features</h2>
<ul>
  <li>Add new Task by clicking onto Floating Button </li>
  <li>Delete Task by swiping task left </li>
  <li>Edit Task by swiping task right </li>
</ul>
